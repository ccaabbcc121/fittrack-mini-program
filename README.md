# 个人健身管理微信小程序
## 📌 项目简介
本项目基于**微信原生小程序 WXML+WXSS+JavaScript**独立开发，是面向个人的健身数据管理工具，当前采用小程序本地缓存存储数据，仅个人自测使用，后续可对接云开发实现云端存储、线上发布。
已解决灰度基础库线程阻塞 `WAServiceMainContext timeout` 兼容性问题，修复新增保存按钮交互异常等前端bug，整体业务闭环可正常运行。

## ✨ 核心功能
1. 首页总览看板：展示当前体重、BMI、本周训练、当日摄入消耗热量统计
2. 体重管理模块：新增/编辑/删除体重记录，记录体重变化幅度
3. 饮食记录模块：早午晚餐录入，统计热量、蛋白质、碳水、脂肪
4. 训练计划模块：自定义训练方案，增删改训练动作与时长热量
5. BMI计算器：快速测算身体质量指数与健康区间评估

## 🛠 技术环境
- 开发工具：微信开发者工具 Stable 2.01
- 小程序基础库：3.5.8（稳定版，修复兼容问题）
- 数据存储：wx.setStorageSync 本地持久化缓存
- 版本管理：Git + GitHub 规范化代码管理
